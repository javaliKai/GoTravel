Font-Awesome

Notice that I use some third-party library called 'font-awesome' to grab some cute icons for me to use in the project. It's all included in the project file. An important thing to be noted, in order for the icons to show up, you need to at least have an internet connection (because that's how font-awesome work, fetching icons through the back-end side) otherwise, the icons won't show up.


Basic Idea

So the basic idea of the project is I wanted to make an online website for a travel agency called 'GoTravel' (and of course it's not real). The website provides general information about GoTravel, and the main feature I wanted to emphasize here is the booking page. The booking page allows the visitor to fill in a form and issue an online booking for travel services.

Styling

For the style, I determined all of them myself which probably you'll find some of it bad (I'm not a good designer btw, but eager to learn). I use a color picker online to generate a color scheme to be used throughout the pages.


As for the font family, I downloaded a google font from fonts.google.com into my local project directory and import it into the CSS.

Now for the CSS properties and layout, you may notice that I use some properties that are not learned during the web programming lecture. For example:

  • I use CSS 'Flexbox' property to adjust the navbar and footer spacing
  • I use CSS variable to assign a custom name for my color scheme
  • I applied a media query to make the website responsive in a mobile device


For the Flexbox, I learned it myself and I understand how it works. It's a cool feature provided by CSS which organize my item more gracefully.

For the variable, I learned it myself during my free time. I need it to keep track of my color scheme, so I don't need to type the hex code over and over again, saves time and more productive.

And yeah, try to resize the website window to see the responsiveness.
